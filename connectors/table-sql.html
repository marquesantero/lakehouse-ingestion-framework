Connector

Table and SQL

Use catalog-native sources when data is already registered in Spark or Unity Catalog.

When to use

Choose table, delta_table or view for registered objects. Choose sql when the source must be a reviewed query.

source:
  type: connector
  connector: table
  table: main.raw.orders
source:
  type: connector
  connector: sql
  query: |
    SELECT id, updated_at, amount
    FROM main.raw.orders
    WHERE status = 'closed'

Operational guidance

  • Use fully qualified names in shared workspaces.
  • Keep SQL short enough to review; complex transformations usually belong in views or Silver tables.
  • Permissions are runtime/catalog concerns, not connector behavior.
